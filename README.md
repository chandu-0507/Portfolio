# Chandu Ummanaveni - Portfolio Website

This repository contains the source code for my personal **portfolio website**, built using **HTML, CSS, and JavaScript**.  
It showcases my profile, technical skills, projects, education, certifications, and contact information.

## 🌐 Live Demo

🔗 **Portfolio Website:** [View Live](https://chandu-0507.github.io/Portfolio/)

## 📖 Overview

The portfolio is designed to present my background as an **Associate Software Engineer** and **Python Full Stack Developer / Data Analytics Enthusiast**.  
It includes sections for:

- Home
- About Me
- Technical Skills
- Projects
- Education & Certifications
- Contact Information

## 🚀 Features

- Responsive portfolio layout
- Hero section with introduction and social links
- About section with professional summary
- Technical skills categorized by domain
- Project showcase section
- Education timeline and certifications
- Contact section with contact form
- Font Awesome icons integration
- Google Fonts styling

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Font Awesome**
- **Google Fonts**
